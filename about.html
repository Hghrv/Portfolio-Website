About this website

On this dedicated platform, you have a presentation of various recent projects including backend to frontend demonstrations.

Linux Control Line Interface is mostly used for the whole programing of this website. A well secure and basic website like this one could effectively be deployed via Data Hosting platforms such as Amazon Web Services, with dedicated modules such as Terraform for the creation of resources and initialisation of permissions for serveless automated applications and orchestrations, Bash scripts for automation, and Moto3 for scalable testing.
